Step 1: Save This HTML File

Download this file and save it to your computer. This is your interactive map component.

Step 2: Add Custom Element in Wix Studio

Method A: Using Custom Element (Recommended)

  1. Open your Wix Studio editor
  2. Click "Add Elements" (+) on the left panel
  3. Go to "Embed & Social" ‚Üí "Custom Element"
  4. Drag the Custom Element to your page
  5. Click "Choose Source" ‚Üí "Website Address (URL)"
  6. Upload this HTML file to a free hosting service (like GitHub Pages, Netlify, or Cloudflare Pages)
  7. Paste the URL into the custom element

Method B: Using Embed Code

  1. Add an "HTML iframe" element from "Embed & Social"
  2. Click "Enter Code" and paste the entire HTML code
  3. Set the iframe size (recommended: width 100%, height 800-1000px)

Step 3: Create Regional Pages

For each region you need to create a separate page in Wix:

  1. Go to "Pages" in your Wix Studio
  2. Create new pages named: scotland, wales, london, etc.
  3. Each page will display campsites for that specific region

Step 4: Set Up Campsite Database (Free Options)

Campsite Data Structure:

{ "name": "Lakeside Retreat", "region": "scotland", "address": "123 Glen Road, Highlands", "what3words": "///lamp.rush.lions", "url": "https://example.com", "rating": 4.5, "status": "visited", // or "favorite" or "wishlist" "ehu": true, "openAllYear": false, "onsite": ["Shower blocks", "Shop", "Laundry"], "nearby": ["Hiking trails", "Fishing", "Village pub"], "comments": "Beautiful location with stunning views", "latitude": 56.8, "longitude": -4.2 }

Storage Options (All Free):

1. Wix Collections

Use Wix's built-in database (CMS). Create a "Campsites" collection with all the fields mentioned above.

2. JSON File

Store data in a JSON file hosted on GitHub or your site. Simple but requires manual updates.

3. Google Sheets

Use Google Sheets API (free) to store and retrieve data. Easy to update, but requires API setup.

4. Airtable

Free tier available. Great interface and easy API integration.
